Project Overview

The project is to create a website that visualizes weather and air data

The website consists of 2 pages:

  • A page that allows you to view the air quality graphics
  • A page that allows you to view the weather graphics
  • A page that allows you to view the graphics
  • A page that allows you to view the map
  • A page that allows you to view the forecast

The website is developed in Python with the Flask framework

Objective

The goal of the project is to create a website that allows you to view weather and air
