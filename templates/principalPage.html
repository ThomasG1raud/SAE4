Présentation du projet

Le projet consiste à créer un site web qui permet de visualiser les données météo et air

Le site web est composé de 2 pages :

  • Une page qui permet de visualiser les données météo et air
  • Une page qui permet de visualiser les graphiques

Le site web est développé en Python avec le framework Flask

Objecif

Le but du projet est de créer un site web qui permet de visualiser les données météo et air
1. Collecte des donnees à partir de deux sources de données :

  • Les données air sont récupérées depuis le site hub.arcgis.com
  • Les données météo sont récupérées depuis le site data.opendatasoft.com
2. Fusion et integration de données. Appliquer les procédures de pré-traitement pour gérer les donnés collectees et les créeer/modifier en utilisant pandas [1].
  • Integration des données: combiner les données provenant de différentes sources dans une vue unifiee
  • Normalisation des variables quantitatives
  • Encodage des variables qualitatives.
3. Production de statistiques descriptives:
  • Pour chaque caracteristique (selon son type)
  • Ou pour l'ensemble de ce jeu de donnees.
4. Chercher les donnees manquantes, et tester plusieurs méthodes de complétion.
  • Evaluation des résultats de différentes méthodes par plusieurs métriques si c'est possible
  • Chercher les meilleurs methodes.
5. Recherche et traitement de donnees aberrantes, par plusieurs maniéres.
  • Comment trouver les meilleures methodes ?
  • Evaluation des résultats par plusieurs mètriques.
  • Quelle methode vous devriez utiliser parmi celles proposées par l'application?
6. Visualisation de donnees pour une caracteristique/variable explicative (selon son type): sélectionner parmi celles proposees par l'application fournie ou pour l'ensemble des caractéristiques (récurrence, distribution, etc).
7. Selection de caractéristiques: utiliser différentes techniques pour la sélection des caractéristiques, telles que les methodes de sélection par filtre univarié et l’algorithme d’élimination récursif [2, 3].
  • Comment trouver les meilleures methodes ?
  • Evaluation des résultats par plusieurs métriques.
  • Quelle methode vous devriez utiliser parmi celles proposées par l'application?
8. Prediction ou classification (suivant le contexte) par plusieurs techniques d'apprentissage automatique (selection de modéle).
9. Interface publique pour montrer les resultats de l'analyse au client.
